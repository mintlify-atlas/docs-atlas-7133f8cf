Sortvis provides interactive visualizations for six fundamental sorting algorithms. Each algorithm is implemented as a TypeScript generator function that yields visualization steps, allowing you to see exactly how the algorithm works in real-time.

| Bubble sort | O(n) | O(n²) | O(1) | Yes |
| Quick sort | O(n log n) | O(n²) | O(log n) | No |
| Merge sort | O(n log n) | O(n log n) | O(n) | Yes |

## Visualization steps

Each algorithm implementation yields typed steps that drive the visualization:

```typescript
type SortStep =
  | { type: 'compare'; indices: [number, number] }
  | { type: 'swap'; indices: [number, number] }
  | { type: 'pivot'; indices: [number] }
  | { type: 'sorted'; indices: number[] }
  | { type: 'merge'; indices: [number]; value: number };
```

- **compare** - Highlight two elements being compared
- **swap** - Exchange two elements
- **pivot** - Mark an element as the pivot (quick sort)
- **sorted** - Mark elements as in their final position
- **merge** - Write a value during the merge phase (merge sort)

## Algorithm metadata

Each algorithm includes comprehensive metadata defined in `src/lib/config/algorithms.ts`:

```typescript
interface AlgorithmInfo {
  id: AlgorithmId;
  name: string;
  minTimeComplexity: string;
  maxTimeComplexity: string;
  spaceComplexity: string;
  stable: boolean;
  description: string;
}
```

This metadata powers the UI display and helps users understand the performance characteristics of each algorithm.
